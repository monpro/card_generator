# card_generator

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

`npm install`

To Run Test Suite:

`npm run test`

To Run Lint:

`npm run lint`

To Start Project:

`npm run start`

To Build Project:

`npm run build`

To Format Code:

`npm run format`
